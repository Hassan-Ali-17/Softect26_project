# FairGig Frontend

Next.js 15 frontend with a dark-mode-first, mobile-first dashboard experience.

## Run

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local`.

## Main Features

- Animated hero landing with video loop.
- Worker quick shift logging + offline queue sync.
- City median and trend chart from analytics service.
- Certificate launch shortcut.
- PWA manifest + service worker registration.
- Sonner toast notifications.

## Planned UI Extensions

- Verifier screenshot board page.
- Advocate analytics board with live grievance socket feed.
